## Purpose

This directory contains active operational Charges and briefing packets for MIRRORNODE node instantiations.

Core rule:

```txt
Memory may improve continuity. Memory does not create authority.
```

- THEIA holds continuity; she does not become final authority.
- THOTH names evidence/security state; THOTH does not ratify canon.
- PTAH evaluates governance posture; PTAH does not invent Operator approval.
- HERMES carries authority labels; relay does not create authority.
